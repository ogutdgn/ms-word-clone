# View Tab — Feature Status

_Verified by 8 tests (154/154 total) + web research. View-type enum (Draft=1/Outline=2/Print=3/Web=6/Reading=7) is the MS Word ground truth._

## Views
- ✅ **Read Mode**, **Print Layout**, **Web Layout**, **Outline** (headings indented by level with markers, body dimmed), **Draft** (continuous, no page sheet)

## Immersive / Page Movement
- ✅ **Focus**, **Immersive Reader** (distraction-free overlay: A−/A+ text size, Sepia/Dark, Read Aloud)
- ✅ **Vertical** / **Side to Side** page movement toggle

## Show / Zoom
- ✅ **Ruler**, **Gridlines**, **Navigation Pane**
- ✅ **Zoom** dialog, **100%**, **One Page**, **Multiple Pages**, **Page Width**

## Window
- ✅ **Split** (a second pane divider)
- ❌ **New Window / Arrange All / View Side by Side / Synchronous Scrolling / Reset Window Position / Switch Windows** — multi-document/multi-window features, not supported in this single-window clone (documented stubs)

## Macros / SharePoint
- ❌ **Macros** (View/Record) — no VBA runtime (documented)
- ✅ **Properties** — document properties dialog (title/author/words/chars/pages/lines)
